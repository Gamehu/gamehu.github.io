ж•ҙдёӘиҝҮзЁӢжҳҜжёҗиҝӣзҡ„иҝҮзЁӢпјҢжөҸи§ҲеҷЁдёҚдјҡзӯүзқҖжүҖжңүhtmlж–ҮжЎЈйғҪи§Јжһҗе®ҢеҶҚе‘ҲзҺ°пјҢиҖҢжҳҜи§ЈжһҗиҝҮзЁӢдёӯе°ұејҖе§Ӣжһ„е»әе‘ҲзҺ°ж ‘е’Ңи®ҫзҪ®еёғеұҖгҖӮеңЁдёҚж–ӯжҺҘж”¶е’ҢеӨ„зҗҶжқҘиҮӘзҪ‘з»ңзҡ„е…¶дҪҷеҶ…е®№зҡ„еҗҢж—¶пјҢе‘ҲзҺ°еј•ж“Һдјҡе°ҶйғЁеҲҶеҶ…е®№и§Јжһҗе№¶жҳҫзӨәеҮәжқҘгҖӮ

дё“дёҡи§ЈйҮҠпјҡHTML з»ҸиҝҮи§Јжһҗз”ҹжҲҗ DOM TreeпјӣиҖҢеңЁ CSS и§Јжһҗе®ҢжҜ•еҗҺпјҢйңҖиҰҒе°Ҷи§Јжһҗзҡ„з»“жһңдёҺ DOM Tree зҡ„еҶ…е®№дёҖиө·иҝӣиЎҢеҲҶжһҗе»әз«ӢдёҖжЈө Render TreeпјҲе‘ҲзҺ°ж ‘пјүпјҢжңҖз»Ҳз”ЁжқҘиҝӣиЎҢз»ҳеӣҫгҖӮRender Tree дёӯзҡ„е…ғзҙ дёҺ DOM е…ғзҙ зӣёеҜ№еә”пјҢдҪҶйқһдёҖдёҖеҜ№еә”пјҡдёҖдёӘ DOM е…ғзҙ еҸҜиғҪдјҡеҜ№еә”еӨҡдёӘ rendererпјҢеҰӮж–Үжң¬жҠҳиЎҢеҗҺпјҢдёҚеҗҢзҡ„гҖҢиЎҢгҖҚдјҡжҲҗдёә render tree з§ҚдёҚеҗҢзҡ„ rendererгҖӮд№ҹжңүзҡ„ DOM е…ғзҙ иў« Render Tree е®Ңе…Ёж— и§ҶпјҢжҜ”еҰӮ display:none зҡ„е…ғзҙ гҖӮ

иҝҷе„ҝйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢhtmlзҡ„и§ЈжһҗжҳҜжӯЈеҗ‘зҡ„д»ҺдёҠеҲ°дёӢпјҲд»Һе·ҰеҲ°еҸіпјүзҡ„пјҲд»Һж №иҠӮзӮ№ејҖе§ӢпјүпјҢиҖҢcssжҳҜйҖҶеҗ‘зҡ„д»ҺдёӢеҲ°дёҠпјҲд»ҺеҸіеҲ°е·Ұпјүзҡ„гҖӮдёәд»Җд№ҲhtmlжҳҜд»ҺдёҠеҲ°дёӢзҡ„пјҢиҝҷдёӘеҫҲеҘҪзҗҶи§ЈпјҢдҪ жғіжғіhtmlи§ЈжһҗеҗҺжҳҜдёҖжЈөж ‘пјҢдҪ жғіжғіж ‘зҡ„жһ„жҲҗиӮҜе®ҡжҳҜе…ҲжңүдёӘдё»е№Із„¶еҗҺеҶҚжңүеҲҶеҸүпјҢиҝҷж ·жүҚжҳҜдёҖдёӘж•ҙдҪ“пјҢдёҚз„¶дёҚе°ұжҳҜдёҖзӣҳж•Јиҙ§д№ҲгҖӮйӮЈcssдёәд»Җд№Ҳе°ұдёҖе®ҡиҰҒд»ҺдёӢеҲ°дёҠе‘ўпјҢйҖҡиҝҮдёӢйқўзҡ„еҸҷиҝ°дҪ дјҡзҹҘйҒ“еҰӮжһңcssд№ҹйҮҮеҸ–д»ҺдёҠеҲ°дёӢзҡ„йЎәеәҸпјҢйӮЈdom treeе’Ңstyle rulesдёӨдёӘйңҖиҰҒеҜ№дёҠзңјзҡ„ж¶ҲиҖ—жҳҜеҫҲеӨ§зҡ„пјҲеӣ дёәйҖҡеёёdomиҠӮзӮ№еӨҡпјҢcssж ·ејҸд№ҹеӨҡпјү,д»ҺжҰӮзҺҮдёҠжқҘи®ІйҖҶеҗ‘еҢ№й…Қзҡ„жҰӮзҺҮжҳҜиҝңиҝңй«ҳдәҺжӯЈеҗ‘еҢ№й…Қзҡ„жҰӮзҺҮзҡ„гҖӮ

дё“дёҡи§ЈйҮҠпјҡеңЁе»әз«Ӣ Render Tree ж—¶пјҢ DOM Tree дёӯзҡ„е…ғзҙ йңҖиҰҒж №жҚ® CSS зҡ„и§Јжһҗз»“жһңпјҲStyle RulesпјүжқҘзЎ®е®ҡз”ҹжҲҗжҖҺж ·зҡ„ rendererгҖӮеҜ№дәҺжҜҸдёӘ DOM е…ғзҙ пјҢеҝ…йЎ»еңЁжүҖжңү Style Rules дёӯжүҫеҲ°з¬ҰеҗҲзҡ„ selector е№¶е°ҶеҜ№еә”зҡ„и§„еҲҷиҝӣиЎҢеҗҲе№¶гҖӮйҖүжӢ©еҷЁзҡ„гҖҢи§ЈжһҗгҖҚе®һйҷ…жҳҜеңЁиҝҷйҮҢжү§иЎҢзҡ„пјҢеңЁйҒҚеҺҶ DOM Tree ж—¶пјҢд»Һ Style Rules дёӯеҺ»еҜ»жүҫеҜ№еә”зҡ„ selectorгҖӮ

еҰӮжһңжӯЈеҗ‘и§ЈжһҗпјҢдҫӢеҰӮгҖҢdiv div p emгҖҚпјҢжҲ‘д»¬йҰ–е…Ҳе°ұиҰҒжЈҖжҹҘеҪ“еүҚе…ғзҙ еҲ° html зҡ„ж•ҙжқЎи·Ҝеҫ„пјҢжүҫеҲ°жңҖдёҠеұӮзҡ„ divпјҢеҶҚеҫҖдёӢжүҫпјҢеҰӮжһңйҒҮеҲ°дёҚеҢ№й…Қе°ұеҝ…йЎ»еӣһеҲ°жңҖдёҠеұӮйӮЈдёӘ divпјҢеҫҖдёӢеҶҚеҺ»еҢ№й…ҚйҖүжӢ©еҷЁдёӯзҡ„з¬¬дёҖдёӘ divпјҢеӣһжәҜиӢҘе№Іж¬ЎжүҚиғҪзЎ®е®ҡеҢ№й…ҚдёҺеҗҰпјҢж•ҲзҺҮеҫҲдҪҺгҖӮ

йҖҶеҗ‘еҢ№й…ҚеҲҷдёҚеҗҢпјҢеҰӮжһңеҪ“еүҚзҡ„ DOM е…ғзҙ жҳҜ divпјҢиҖҢдёҚжҳҜ selector жңҖеҗҺзҡ„ emпјҢйӮЈеҸӘиҰҒдёҖжӯҘе°ұиғҪжҺ’йҷӨгҖӮеҸӘжңүеңЁеҢ№й…Қж—¶пјҢжүҚдјҡдёҚж–ӯеҗ‘дёҠжүҫзҲ¶иҠӮзӮ№иҝӣиЎҢйӘҢиҜҒгҖӮе…¶е®һжӣҙж·ұе…Ҙзҡ„зҗҶи§ЈйңҖиҰҒж·ұе…ҘдәҶи§ЈHtml Parserе’Ңcss Parserзҡ„е·ҘдҪңиҝҮзЁӢгҖӮ
